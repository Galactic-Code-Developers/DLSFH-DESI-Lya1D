# DESI Lyα 1D data

This folder contains DESI Lyα 1D power spectrum products as provided by the user workflow.

- `datapoints_p1d_fft_desi_edrm2_v1.zip`:
  upstream DESI product bundle used to extract P1D points.

- Mask lists:
  text files used to select or mask k-bins according to the upstream extraction workflow.

If you redistribute this repository publicly, verify DESI data product terms.
